## Instalasi

1. Clone repositori ini:
   ```bash
   git clone https://github.com/autoresbot/terminal.git
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd terminal
   ```
3. Instal dependensi:
   ```bash
   npm install
   ```
4. Jalankan aplikasi:
   ```bash
   node index.js
   ```
